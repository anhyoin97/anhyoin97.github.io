# Node-RED 시작하기 (Mac 기준)

## 왜 Node-RED인가?
Node-RED는 드래그 앤 드롭 방식으로 데이터 흐름을 구성할 수 있는 로우코드(Low-code) 도구다.  
MQTT, HTTP, DB, 파일, API 등 다양한 시스템을 빠르게 연결할 수 있어 `PoC(개념 검증)`나 `시스템 연계 구조 파악`에 매우 적합하다.

특히 짧은 시간 안에 이 가능하다는 점이 장점이다.
- 데이터 흐름 전체 구조 이해
- 여러 시스템을 잇는 허브 역할 체험
- 시각적으로 설명 가능한 결과물 생성

---

## 개발 환경
- macOS
- Node.js v25.2.1
- npm v11.7.0

---

## Node.js 설치 확인
```bash
node -v
npm -v
```

## Node-RED 설치
```bash
npm install -g node-red
```

## Node-RED 실행
```bash
node-red
```

터미널에 아래와 같은 로그가 출력되면 정상 실행이다.
```bash
Server now running at http://127.0.0.1:1880/
```

## 브라우저 접속
```bash
http://127.0.0.1:1880/
```

## 정리
- Node-RED는 Node.js 위에서 동작한다
- 설치와 실행이 매우 간단하다
- 브라우저 기반 UI로 데이터 흐름을 직관적으로 구성할 수 있다
